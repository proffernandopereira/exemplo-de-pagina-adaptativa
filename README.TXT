Descrição

    HTML Estruturado: A estrutura HTML é simples, com um contêiner principal que contém um cabeçalho, uma seção de conteúdo com três caixas e um rodapé.
    Estilos Básicos: O CSS define estilos básicos para o layout e as caixas de conteúdo.
    Media Queries: O CSS inclui media queries para criar layouts adaptativos:
        Para telas com largura máxima de 1200px, as caixas (divs) se ajustam para ocupar 45% da largura.
        Para telas com largura máxima de 768px, as caixas se ajustam para ocupar 100% da largura.
        Para telas com largura máxima de 480px, o cabeçalho e o rodapé ajustam seu padding e tamanho da fonte, e as caixas de conteúdo ajustam seu padding.

Neste exemplo, cada media query define um layout específico para diferentes tamanhos de tela, 
garantindo que a página se adapte de maneira apropriada conforme o dispositivo utilizado.
